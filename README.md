# 臨床統計ラボ

## テーマ
臨床試験の統計解析手法をSAS/Rのコード付きで実践的に解説するブログ。統計解析計画書（SAP）の書き方から、個別の解析手法、規制当局が求める統計要件まで幅広くカバーする。

## ターゲット読者
- 製薬企業・CROの生物統計担当者
- 臨床試験データを扱うSAS/Rプログラマー
- 統計解析の基礎を学びたい臨床開発担当者

## 想定記事カテゴリ
- SAS実践テクニック（PROC MIXED、PROC LIFETEST等）
- R for Clinical Trials（tidyverse、admiral、rtables）
- 統計解析計画書（SAP）作成ガイド
- 生存時間解析・混合効果モデル等の手法解説
- ICH E9(R1) Estimandフレームワーク
